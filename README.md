# 微信小程序商城
微信小程序商城demo，风格模仿xx严选，简约美观、现代化

## 简介
使用 TouchUI （http://www.wetouch.net） 框架
IDE 使用 VSCode
使用TouchUI框架可以一键转换成h5版本

## 说明
只做了前端样式，数据都是写死模拟的，页面不是非常完善，大部分有了
部分图片源自网易严选，仅供学习参考
